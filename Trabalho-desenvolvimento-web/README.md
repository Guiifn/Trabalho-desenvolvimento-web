# Trabalho-desenvolvimento-web
Roda & Sabor – Restaurante Gamificado

Sobre o projeto

O Roda & Sabor é um sistema web de um restaurante fictício que utiliza gamificação para oferecer descontos aos clientes. A proposta do projeto é criar uma experiência divertida e interativa onde os usuários podem ganhar cupons de desconto através de jogos, como uma roleta de prêmios.

A ideia é transformar a forma tradicional de receber promoções em algo dinâmico e envolvente, incentivando a interação dos usuários com a plataforma.

O sistema permite que clientes criem uma conta, participem dos jogos disponíveis e recebam cupons exclusivos de desconto para utilizar no restaurante fictício.

---

Objetivo

O objetivo do projeto é desenvolver uma aplicação web completa utilizando arquitetura com front-end, back-end e API, demonstrando conceitos importantes do desenvolvimento moderno, como:

- Autenticação de usuários
- Consumo e criação de APIs
- Persistência de dados em banco de dados
- Lógica de negócio para geração de cupons
- Gamificação aplicada a sistemas web

---

Funcionalidades principais

- Cadastro e login de usuários
- Sistema de roleta de prêmios
- Geração automática de cupons de desconto
- Histórico de cupons do usuário
- Controle de tentativas de jogo
- Interface simples e intuitiva simulando um restaurante online

---

Estrutura do sistema

O projeto é dividido em três partes principais:

Front-end

- Interface do usuário
- Interação com os jogos
- Visualização de cupons e informações

Back-end

- Lógica do sistema
- Autenticação de usuários
- Controle de jogos e geração de cupons

API

- Comunicação entre front-end e back-end
- Endpoints para usuários, jogos e cupons

---

Diferencial do projeto

O principal diferencial do Roda & Sabor é a aplicação de gamificação em um sistema de restaurante, permitindo que usuários conquistem benefícios através de jogos simples e interativos.

Essa abordagem busca simular estratégias modernas de marketing digital utilizadas por empresas para engajar clientes e aumentar a retenção.
